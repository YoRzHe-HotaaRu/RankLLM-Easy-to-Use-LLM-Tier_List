# RANK/LLM

A sharp, light-theme drag-and-drop tier list for 2026 frontier LLMs.

## Run

```bash
npm install
npm run dev
```

## Features

- Tiers S through D plus an unranked model bench
- Drag models between tiers and the bench
- Search and provider filters
- Reset, chaos-rank, and markdown export
- LocalStorage persistence

## Stack

Vite · React · TypeScript · @dnd-kit
